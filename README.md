ubuntu docker image
=============================

A slim ubuntu image for docker

packages
-----------------------------

* vim
* curl
* net-tools
* git
* locales

settings
-----------------------------

* en_US.utf8 locale


notes
----------------------------

the image is created in one layer and all temporary files are removed in order to keep it 
low in size
